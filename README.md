# Study Spring

Spring Framework와 Spring Boot의 핵심 개념 및 기술을 학습 및 복습

## 📌 목표

* Spring의 전반적인 동작 원리 이해
* MVC 패턴 기반 웹 애플리케이션 개발 학습
* DI, IoC, AOP 등 핵심 개념 복습
* JDBC, JPA를 활용한 데이터베이스 연동
* REST API 개발 및 테스트
* Spring Boot 기반 프로젝트 구조 학습

Spring 관련 기술을 실습 중심으로 학습 및 복습하며,
